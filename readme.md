# EEG Emotion Detection Playground
This repositry contains codes and datasets for eeg emotion detection.

# Code Names
- BC : Baseline Correction
- BR : Baseline Removal
- Tx : segment length is x seconds
- Ox : overlp is x seconds
- Grid : ToGrid()
- 2d : To2d()
